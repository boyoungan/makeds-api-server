# MCP API 서버

MCP(Management Control Platform) 애플리케이션을 위한 API 서버입니다. 이 서버는 Node.js와 Express를 기반으로 하며, PostgreSQL 데이터베이스를 사용하는 백엔드 시스템입니다.

## 기능

- 사용자 인증 및 권한 관리 (로그인, 회원가입, 비밀번호 변경)
- 물품 관리 API
- 구매 요청 관리 API
- 구매 주문 관리 API
- 휴가 계산 API
- 문서 채팅 기능 (AI 기반)
- 프롬프트 라이브러리 관리
- 문서 유형 변환 기능
- 퀴즈 매뉴얼 API
